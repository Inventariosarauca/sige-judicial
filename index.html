Configuración de Firebase Firestore

Este sistema usa Firebase Firestore (plan gratuito Spark) como base de datos. Sigue estos pasos para conectar tu propia instancia:

1
Ve a console.firebase.google.com e inicia sesión con tu cuenta Google.
2
Crea un nuevo proyecto → Habilita Firestore Database en modo de producción o prueba.
3
Ve a Configuración del proyecto → Aplicaciones web → Agrega una app web y copia la configuración firebaseConfig.
4
En Firestore → Reglas, permite lecturas y escrituras (para pruebas):
// Reglas permisivas para desarrollo:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
5
Pega tu configuración JSON en el campo de abajo y haz clic en Conectar.
El plan gratuito Spark incluye 50.000 lecturas y 20.000 escrituras diarias — más que suficiente para uso institucional moderado.

Publicar en GitHub Pages

Para alojar este sistema gratuitamente en GitHub Pages:

1
Crea un repositorio en github.com (puede ser privado).
2
Sube este archivo HTML al repositorio como index.html.
3
Ve a Settings → Pages del repositorio y habilita GitHub Pages desde la rama main.
4
Tu sistema estará disponible en https://tu-usuario.github.io/tu-repo en minutos.
GitHub Pages es completamente gratuito para repositorios públicos. Para repositorios privados con Pages, necesitas un plan de pago de GitHub.

Exportación para Excel

Usa el botón "Exportar CSV" en la sección de Órdenes de Servicio. El archivo CSV puede abrirse directamente en Excel o usarse con macros VBA para procesamiento avanzado.

' Macro VBA para importar CSV de SIGE a Excel:
Sub ImportarCSV()
  Dim ruta As String
  ruta = Application.GetOpenFilename("CSV,*.csv")
  If ruta = "False" Then Exit Sub
  Workbooks.Open ruta
End Sub
