Configuración de Firebase Firestore

Conecta tu base de datos Firebase Firestore para que los datos se guarden en la nube y sean accesibles desde cualquier dispositivo. Si no configuras Firebase, los datos se guardan solo en este navegador.

Firebase Firestore
  • Datos en la nube
  • Acceso desde cualquier PC
  • Sincronización en tiempo real
  • Plan Spark: 100% gratuito
Modo Local
  • Solo en este navegador
  • Sin internet requerido
  • Exportar CSV para respaldo
  • Listo para usar ya
Pasos para obtener tu configuración Firebase:
1. Ve a console.firebase.google.com → Crea proyecto → Habilita Firestore Database
2. Firestore → Reglas → pega: allow read, write: if true; → Publicar
3. Configuración del proyecto → Aplicaciones web → Registrar app → Copia el objeto firebaseConfig
Publicar en GitHub Pages (opcional)

Para que el sistema esté disponible desde cualquier navegador sin instalar nada:

1. Ve a github.com → Crear repositorio público → "sige-ac-judicial"
2. Sube este archivo HTML como index.html
3. Settings → Pages → Branch: main / root → Save
4. URL disponible en: https://tu-usuario.github.io/sige-ac-judicial/
